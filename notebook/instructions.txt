Advanced Deployment Instructions
================================

Terraform group 'primary' was successfully created in directory notebook/primary
To deploy, run the following commands:

terraform -chdir=notebook/primary init
terraform -chdir=notebook/primary validate
terraform -chdir=notebook/primary apply

Destroying infrastructure when no longer needed
===============================================

Automated
---------

./ghpc destroy notebook

Advanced / Manual
-----------------
Infrastructure should be destroyed in reverse order of creation:

terraform -chdir=notebook/primary destroy
