> **_NOTE:_** Cluster Toolkit is dropping support for the external [Google Cloud DAOS](https://github.com/daos-stack/google-cloud-daos/tree/main) repository. The DAOS example blueprints (`hpc-slurm-daos.yaml` and `pfs-daos.yaml`) have been removed from the Cluster Toolkit. We recommend migrating to the [Managed Lustre](../../../../modules/file-system/managed-lustre/) module for high-performance parallel file system needs. To help with this transition, see the Managed Lustre example blueprints ([pfs-managed-lustre-slurm.yaml](../../../../examples/pfs-managed-lustre-slurm.yaml) and [gke-managed-lustre.yaml](../../../../examples/gke-managed-lustre.yaml)). If the external [Google Cloud DAOS](https://github.com/daos-stack/google-cloud-daos/tree/main) repository is necessary, we recommend using the last Cluster Toolkit [v1.41.0](https://github.com/GoogleCloudPlatform/cluster-toolkit/releases/tag/v1.41.0).
