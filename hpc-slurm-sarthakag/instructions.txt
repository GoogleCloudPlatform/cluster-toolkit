Advanced Deployment Instructions
================================

Terraform group 'primary' was successfully created in directory hpc-slurm-sarthakag/primary
To deploy, run the following commands:

terraform -chdir=hpc-slurm-sarthakag/primary init
terraform -chdir=hpc-slurm-sarthakag/primary validate
terraform -chdir=hpc-slurm-sarthakag/primary apply

Destroying infrastructure when no longer needed
===============================================

Automated
---------

gcluster destroy hpc-slurm-sarthakag

Advanced / Manual
-----------------
Infrastructure should be destroyed in reverse order of creation:

terraform -chdir=hpc-slurm-sarthakag/primary destroy
