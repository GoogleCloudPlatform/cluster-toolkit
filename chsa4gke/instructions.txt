Advanced Deployment Instructions
================================

Terraform group 'primary' was successfully created in directory chsa4gke/primary
To deploy, run the following commands:

terraform -chdir=chsa4gke/primary init
terraform -chdir=chsa4gke/primary validate
terraform -chdir=chsa4gke/primary apply

Destroying infrastructure when no longer needed
===============================================

Automated
---------

gcluster destroy chsa4gke

Advanced / Manual
-----------------
Infrastructure should be destroyed in reverse order of creation:

terraform -chdir=chsa4gke/primary destroy
