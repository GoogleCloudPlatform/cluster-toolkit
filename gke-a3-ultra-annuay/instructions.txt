Advanced Deployment Instructions
================================

Terraform group 'primary' was successfully created in directory gke-a3-ultra-annuay/primary
To deploy, run the following commands:

terraform -chdir=gke-a3-ultra-annuay/primary init
terraform -chdir=gke-a3-ultra-annuay/primary validate
terraform -chdir=gke-a3-ultra-annuay/primary apply

Destroying infrastructure when no longer needed
===============================================

Automated
---------

gcluster destroy gke-a3-ultra-annuay

Advanced / Manual
-----------------
Infrastructure should be destroyed in reverse order of creation:

terraform -chdir=gke-a3-ultra-annuay/primary destroy
